# Oky

`oky-validators` use `ramda` and `ramda-validations` for validators.

## Validators List

### General

- isEqualsTo
- isPresent
- isBlank
- isOtherThan

### Number

- isGreaterThan
- isGreaterThanOrEqualTo
- isLessThan
- isLessThanOrEqualTo
- isEven
- isOdd
- isNegative
- isPositive
- isInteger
- isFloat
- isFinite
- isMultiple

## String

- hasPattern
- isAlphanum
- isString
- isAlphanum

## Boolean

- isTrue
- isFalse

## Array

- isContains
- isInRange

### Object

- hasKey
- hasKeys
- isObject
- isHash
