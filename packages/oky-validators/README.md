# Oky-validators

`oky-validators` is a list of validators for [oky](https://www.npmjs.com/package/oky).

## Validators List

### General

- isEqualsTo
- isPresent
- isBlank
- isOtherThan

### Number

- isGreaterThan
- isGreaterThanOrEqualTo
- isLessThan
- isLessThanOrEqualTo
- isEven
- isOdd
- isNegative
- isPositive
- isInteger
- isFloat
- isFinite
- isMultiple

## String

- hasPattern
- isAlphanum
- isString
- isAlphanum

## Boolean

- isTrue
- isFalse

## Array

- isContains
- isInRange

### Object

- hasKey
- hasKeys
- isObject
- isHash

# Dependencies

- [oky](https://www.npmjs.com/package/oky)
- [ramda](https://www.npmjs.com/package/ramda)
- [ramda-validations](https://www.npmjs.com/package/ramda-validations)
